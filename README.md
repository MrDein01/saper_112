# Сапёр (Minesweeper) на Pygame

Классическая игра "Сапёр", написанная на Python с использованием библиотеки Pygame. Проект включает уникальные модификаторы, систему рекордов и несколько уровней сложности.

## Основные возможности

- Горячие клавиши: R — быстрый рестарт, 1/2/3/4 — переключение сложности
- Система рекордов: лучшее время сохраняется в файл и загружается при запуске
- Уровни сложности: Легкий, Средний, Сложный, Эксперт
- Модификаторы, меняющие геймплей

## Технологии

- Python 3.x
- Pygame

## Установка и запуск

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/username/repository.git
   ```
2. Перейди в папку проекта:
   ```bash
   cd repository
   ```
3. Установи Pygame:
   ```bash
   pip install pygame
   ```
4. Запусти игру:
   ```bash
   python main.py
   ```
# Как играть
Левая кнопка мыши — открыть клетку

Правая кнопка мыши — поставить/убрать флажок

Цель — открыть все клетки без мин

# Модификаторы
Без модификатора — обычная игра без изменений

Перемешивание — каждые 30 секунд мины меняются местами, флажки остаются на своих местах

АнтиЦепная реакция — Клетки отрыкваются по одиночке

Сапёр-сапёр — раз в 40 секунд появляется возможность обезвредить мину, указав её расположение

Мега-мина — одна мина занимает область 2×2 клетки

Заморозка — каждые 20 секунд флаги блокируются на 5 секунд

Искажение — каждые 25 секунд поле поворачивается на 90°

Темнота — видны только клетки вокруг уже открытых
